A wireless sensor network (WSN) is a collection of spatially distributed autonomous sensor nodes that can be used to monitor, among other things, environmental conditions. WSN sensor nodes are constrained by their limited energy supply, communication range and local computational capabilities. Data routing is an area that can be optimized to allow nodes to conserve energy, improving the network
